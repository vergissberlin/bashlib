You wanna create your own bash script, and don't start on ground zero?
That library is made for you! It helps to improve bash your bash scripts.

## Including

1. UI components
  - Colored messages ``[ OK ]``
	- Progress annimation ``[ / ]``
2. Testing components
3. Validation helper
4. Math helper
5. Remote ssh command execution
